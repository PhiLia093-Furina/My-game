# My-game
练习制作的小游戏

基础依赖
Python 3.x 版本
Pygame 库（需提前安装：pip install pygame）

│   ├── ico2.png          # 玩家2角色贴图

游戏操作
1. 玩家 1 操作（键盘 + 鼠标）
移动：
键盘：方向键（↑↓←→）控制上下左右移动，每次移动一格（40 像素）
鼠标：点击鼠标左键，角色会平滑移动到鼠标点击位置
射击：空格键（Space）发射子弹，子弹方向为角色最后一次移动的方向
传送：触碰传送门（HD 贴图）会随机传送到其他传送门位置
2. 玩家 2 操作（仅键盘）
移动：W（上）、S（下）、A（左）、D（右）控制上下左右移动，每次移动一格（40 像素）
射击：J 键发射子弹，子弹方向为角色最后一次移动的方向
传送：触碰传送门（HD 贴图）会随机传送到其他传送门位置
3. 通用操作
窗口调整：支持窗口大小自由调整（鼠标拖拽窗口边缘）
退出游戏：点击窗口关闭按钮（×）

游戏规则
1. 得分机制
射击命中敌人：得分 + 1
角色推动敌人到指定位置（绿色区域）：得分 + 1
敌人被击败后会自动刷新，保证场上至少 6 个敌人
2. 胜利条件
玩家 1 或玩家 2 得分达到 2 分后，触发通关结算，分数重置并显示通关提示，点击鼠标可继续游戏
3. 碰撞规则
墙体碰撞：角色 / 子弹触碰墙体后，子弹会消失，角色会退回原位置
敌人碰撞：角色触碰敌人后，会尝试推动敌人，若敌人无法移动（撞墙 / 其他敌人），角色退回原位置
传送门碰撞：角色 / 敌人触碰传送门会随机传送到其他传送门位置
4. 敌人 AI
敌人每 40 帧随机向上下左右方向移动一格
敌人移动时若撞墙 / 撞其他敌人 / 撞玩家，会退回原位置并随机更换移动方向
敌人数量不足 6 个时会自动刷新

注意事项
运行前需确保 config/ 和 image/ 文件夹下的贴图文件存在，否则会触发图片加载错误
字体路径 C:/Windows/Fonts/simhei.ttf 为 Windows 系统黑体字体，非 Windows 系统需替换为对应系统的字体路径（如 Linux 的 /usr/share/fonts/truetype/dejavu/DejaVuSans.ttf）
游戏帧率固定为 60 帧 / 秒，保证移动和动画的流畅性
子弹射击有冷却（2 帧），避免高频射击
